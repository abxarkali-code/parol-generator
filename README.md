# parol-generator
Генератор надежных паролей на Python
### Шаг 1. Запустите скрипт
### шаг 2. Введите желаемую длину пароля
### шаг 3. ответьте на вопросы (можно вводить y или д)
### Шаг 4. Получите готовый пароль
### Шаг 5. Сохрани
Листай вниз, найди зеленую кнопку **"Commit changes"** и нажми её.
### Шаг 6. Готово!
Теперь на главной странице код будет выглядеть как в учебнике или в PyCharm — строчка под строчкой, с отступами.
